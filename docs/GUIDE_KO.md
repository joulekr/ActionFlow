# ActionFlow 사용 가이드

ActionFlow는 Windows용 런처 & 생산성 도구입니다. 단축키 하나로 앱 실행, 계산, 단위 변환, 클립보드 관리, 스니펫, 파일 검색 등을 빠르게 수행할 수 있습니다.

## 1. 시작하기

| 동작 | 방법 |
|------|------|
| 런처 열기 / 닫기 | `Ctrl + Shift + Space` |
| 닫기 | `Esc` |

런처를 열면 검색창이 자동으로 포커스됩니다. 검색어를 바로 입력하면 됩니다.

## 2. 기본 모드 (런처)

런처를 열면 기본 모드입니다. 검색어를 입력하지 않으면 자주 실행한 앱을 **최근 실행** 목록으로 자동 표시합니다.

### 2-1. 앱 실행 & 창 전환

```
chrome          → Chrome 실행 (또는 열린 창 활성화)
excel           → 앱 이름 일부로 검색
```

- 이미 열려 있는 창은 앱 검색 결과 위에 함께 표시됩니다.
- 앱을 **우클릭**하면 즐겨찾기에 바로 추가할 수 있습니다.
- 사용 기록이 쌓이면 자주 쓴 앱이 상단에 자동 정렬됩니다 (frecency 기반).

### 2-2. 계산기

```
2 + 3 * 4        → 14
2^10             → 1024
sqrt(16)         → 4
log(1000)        → 3  (상용로그 log₁₀)
ln(1)            → 0  (자연로그)
sin(pi/2)        → 1  (라디안)
abs(-7)          → 7
ceil(1.2)        → 2  (올림)
floor(1.9)       → 1  (내림)
```

결과를 `Enter`로 선택하면 클립보드에 복사됩니다.

지원 연산자: `+ − * / ^ %` 및 괄호 `()`  
지원 함수: `sqrt` `sin` `cos` `tan` `log` `ln` `abs` `ceil` `floor`  
지원 상수: `pi` `e`

### 2-3. 단위 변환

```
100 km to miles  → 62.137 miles
70 kg to lbs     → 154.32 lbs
37 c to f        → 98.60 °F
1 gal to l       → 3.785 l
500 ml to fl_oz  → 16.907 fl_oz
```

형식: `숫자 단위 to 단위` 또는 `숫자 단위 in 단위`

### 2-4. 환율 변환

```
100 usd to krw   → 실시간 환율로 원화 변환
50 eur in jpy    → 유로 → 엔화
1 gbp to usd     → 파운드 → 달러
```

ISO 4217 3자리 통화 코드를 사용합니다. 환율은 1시간 캐시됩니다.

### 2-5. 진법 변환

```
0xFF             → 255 (DEC) · 0b11111111 (BIN) · 0o377 (OCT)
0b11001100       → 0xCC (HEX) · 204 (DEC) · 0o314 (OCT)
255 to hex       → 0xFF
255 to bin       → 0b11111111
```

각 항목 선택 시 해당 표현이 클립보드에 복사됩니다. 지원 범위: 0 ~ 4,294,967,295 (32비트)

### 2-6. 웹 검색

```
?오늘 날씨       → Google 검색 열기
?EPL 경기 결과   → 앞에 ? 를 붙이면 웹 검색
```

### 2-7. 시스템 리소스 모니터

```
cpu              → CPU 사용률 실시간 표시
ram 또는 메모리   → RAM 사용량 표시
disk 또는 디스크  → C 드라이브 사용량 표시
```

### 2-8. 시스템 명령

> 아래 명령은 실행 전 **Enter를 한 번 더** 눌러 확인해야 합니다.

```
lock 또는 잠금                    → 화면 잠금
sleep 또는 절전                   → 절전 모드
close all apps 또는 모든 앱 닫기   → 열려 있는 앱 창 닫기
empty trash 또는 휴지통 비우기     → 휴지통 비우기
restart 또는 재시작                → PC 재시작
shutdown 또는 종료                 → PC 종료
logout 또는 로그아웃               → 로그아웃
kill chrome 또는 종료 notepad      → 실행 중인 프로세스 강제 종료
```

### 2-9. 타이머

```
timer 30s        → 30초 타이머
timer 5m         → 5분 타이머
timer 1h30m      → 1시간 30분 타이머
타이머 5분        → 한국어로도 입력 가능
cancel 또는 취소  → 실행 중인 타이머 취소
```

타이머 완료 시 런처가 다시 표시되며 초록색 알림이 나타납니다.

## 3. 모드 전환

검색창에 `/키워드`를 입력하거나 `Tab` / `Shift+Tab`으로 모드를 전환합니다.

| 입력 | 모드 |
|------|------|
| `/launch` 또는 `/런처` | ⚡ 런처 (기본) |
| `/recent` 또는 `/최근` | 🕒 최근 통합 검색 |
| `/clip` 또는 `/클립` | 📋 클립보드 히스토리 |
| `/snip` 또는 `/스니펫` 또는 `/조각` | 📌 스니펫 |
| `/link` 또는 `/favorite` 또는 `/링크` 또는 `/즐겨찾기` | ⭐ 즐겨찾기 |
| `/file` 또는 `/파일` | 📁 파일 검색 |
| `/emoji` 또는 `/이모지` | 😀 이모지 검색 |
| `/setting` 또는 `/config` 또는 `/설정` | ⚙️ 설정 |

상단 모드 바를 클릭해서 전환할 수도 있습니다.

## 4. 최근 통합 검색

클립보드, 즐겨찾기, 파일의 최근 항목을 한 화면에서 검색합니다.

- 검색어 없이 진입하면 각 소스의 최근 5개 항목을 섹션별로 표시합니다.
- 검색어를 입력하면 3개 소스를 동시에 검색하고, 결과가 있는 섹션만 표시합니다.
- `Enter`로 실행 시 항목 종류에 맞게 동작합니다 (클립보드 → 붙여넣기, 파일 → 실행).

> 파일 검색은 2글자 이상 입력 시 활성화됩니다.

## 5. 클립보드 히스토리

텍스트를 복사하면 자동으로 기록됩니다. 최대 200개 저장, 앱 재시작 후에도 유지됩니다.

### 주요 동작

| 동작 | 방법 |
|------|------|
| 항목 붙여넣기 | `Enter` 또는 클릭 → 이전 창에 자동 붙여넣기 |
| 항목 고정 | `📌` 버튼 → 전체 삭제 시에도 유지, 목록 상단 고정 |
| 항목 수정 | `✏️` 버튼 → 인라인 편집, `Ctrl+Enter`로 저장 |
| 항목 삭제 | `✕` 버튼 |
| 미고정 항목 전체 삭제 | **전체 삭제** 버튼 (고정 항목은 유지) |
| 모니터링 일시정지 | `⏸` 버튼 → 새 복사 내용이 기록되지 않음 |
| 내용 검색 | 검색창에 텍스트 입력 |

> 보관 기간과 최대 저장 개수는 설정에서 변경할 수 있습니다 (기본: 30일, 200개).

## 6. 스니펫

자주 쓰는 텍스트를 트리거 단어로 저장해두고 빠르게 불러옵니다.

### 기본 사용법

1. `/snip`으로 스니펫 모드 진입
2. `+ 새 스니펫` 버튼으로 트리거와 내용 입력
3. 기본 모드에서 `:트리거`를 입력하면 바로 검색

```
:이름    → 저장해둔 이름 텍스트 붙여넣기
:서명    → 이메일 서명 텍스트 붙여넣기
:주소    → 주소 텍스트 붙여넣기
```

> 트리거는 `:`으로 시작하는 것을 권장합니다.

### 내장 스니펫 변수

스니펫 내용에 다음 변수를 삽입하면 활성화 시 자동으로 치환됩니다.

| 변수 | 치환 값 |
|------|---------|
| `{{today}}` 또는 `{{날짜}}` | 오늘 날짜 (YYYY-MM-DD) |
| `{{time}}` 또는 `{{시간}}` | 현재 시간 (HH:MM) |
| `{{datetime}}` | 날짜 + 시간 |
| `{{uuid}}` | 랜덤 UUID |

### 스니펫 관리

| 동작 | 방법 |
|------|------|
| 수정 | `✏️` 버튼 |
| 삭제 | `🗑` 버튼 |
| 내보내기 | **내보내기** 버튼 (JSON) |
| 가져오기 | **가져오기** 버튼 (JSON) |

## 7. 즐겨찾기 (링크)

자주 쓰는 앱, 폴더, 링크, 워크스페이스를 등록하고 빠르게 실행합니다.

### 항목 추가

| 버튼 | 설명 |
|------|------|
| `+ 앱 추가` | 설치된 `.exe` / `.lnk` 파일 선택 |
| `+ 폴더 추가` | 자주 쓰는 폴더 선택 |
| `+ 새 링크` | 이름과 URL 직접 입력 |
| `+ 워크스페이스` | 여러 앱·폴더·링크를 묶은 그룹 생성 |

> URL 입력 시 `https://` 생략 가능합니다.

### 다른 방법으로 추가

- **런처 모드**에서 앱을 우클릭 → ⭐ 즐겨찾기 추가
- **파일 모드**에서 폴더·exe·lnk 항목을 우클릭 → ⭐ 즐겨찾기 추가

### 항목 관리

| 동작 | 방법 |
|------|------|
| 실행 | `Enter` 또는 클릭 |
| 이름 수정 | `✏️` 버튼 |
| 삭제 | `🗑` 버튼 |
| 고정 | `📌` 버튼 → 목록 상단 유지 |
| 순서 변경 | `⠿` 핸들 드래그 |

### 워크스페이스

여러 항목을 묶어 `Enter` 한 번으로 전체를 동시 실행합니다. 프로젝트 시작 시 관련 앱·폴더·URL을 한번에 여는 용도로 활용하세요.

## 8. 파일 검색

바탕화면, 문서, 다운로드 폴더의 파일·폴더를 이름으로 검색합니다.

- 2글자 이상 입력 시 검색이 시작됩니다.
- 파일 크기와 수정일이 함께 표시됩니다.

### 파일 모드 단축키

| 단축키 | 동작 |
|--------|------|
| `Enter` | 파일 실행 / 폴더 탐색기에서 열기 |
| `Space` | 파일 미리보기 토글 |
| `Ctrl+C` | 선택 파일의 경로를 클립보드에 복사 |
| `Ctrl+O` | 해당 파일이 있는 폴더를 탐색기에서 열기 |
| `Ctrl+Enter` | 파일 속성 창 열기 |
| 우클릭 | 폴더 열기 / 경로 복사 / 속성 / ⭐ 즐겨찾기 추가 |

### 전체 폴더 검색

파일 모드 좌측 상단의 **전체 폴더 검색** 토글을 켜면 모든 드라이브를 인덱싱합니다.

- 최초 실행 시 인덱싱에 시간이 걸릴 수 있습니다.
- Windows, Program Files, $Recycle.Bin, AppData\Temp 등 시스템 폴더는 제외됩니다.
- 인덱싱 완료 전까지는 기본 3개 폴더(바탕화면, 문서, 다운로드)에서 검색합니다.

## 9. 이모지 검색

```
/emoji 또는 /이모지  → 이모지 모드 진입
heart               → ❤️ 🧡 💛 … 관련 이모지
fire                → 🔥 관련 이모지
```

- 검색어 없이 열면 자주 쓰는 이모지 목록이 표시됩니다.
- `Enter` 또는 클릭 시 이모지가 클립보드에 복사되고 이전 창에 붙여넣기됩니다.
- 방향키 `←→↑↓`로 그리드 이동, `Enter`로 선택합니다.

## 10. 플러그인

TOML 파일로 나만의 명령을 만들어 런처에서 바로 실행할 수 있습니다.

### 플러그인 폴더 위치

```
%APPDATA%\actionflow\plugins\
```

설정 → **플러그인 폴더 열기** 버튼으로 바로 열 수 있습니다.

> `.toml` 파일을 저장하면 런처에 **자동으로 반영**됩니다 (재시작 불필요).

### 플러그인 파일 형식

```toml
[[commands]]
trigger = "google,구글"          # 트리거 (쉼표로 여러 별칭 가능)
name = "🔍 Google 검색"          # 런처에 표시될 이름
description = "google <검색어>"  # 부연 설명 (선택)
action = "open"                  # 동작 종류
path = "https://www.google.com/search?q={{query}}"
output = "none"
```

### action 종류

| action | 설명 |
|--------|------|
| `shell` | PowerShell 명령 실행 (stdout → output 처리) |
| `open` | 파일 / 앱 / URL 열기 |
| `copy` | 텍스트를 클립보드에 복사 |
| `http` | HTTP GET/POST 요청 (결과를 output 처리) |

### output 종류

| output | 설명 |
|--------|------|
| `clipboard` | 결과를 클립보드에 복사 |
| `paste` | 클립보드 복사 후 이전 창에 붙여넣기 |
| `none` | 실행만 하고 결과 무시 |

### 추가 옵션

| 옵션 | 설명 |
|------|------|
| `confirm = true` | 실행 전 Enter를 한 번 더 요구 (위험한 명령용) |
| `timeout_secs = 10` | 실행 제한 시간 (기본 15초) |
| `cwd = "C:\\경로"` | shell 명령의 작업 디렉토리 |

### 동적 쿼리 `{{query}}`

트리거 뒤에 입력한 텍스트가 `{{query}}`로 전달됩니다.

```
입력: google claude
→ https://www.google.com/search?q=claude 열기
```

### 예시

**PowerShell 명령으로 현재 시간 복사**
```toml
[[commands]]
trigger = "time,시간"
name = "🕐 현재 시간"
action = "shell"
command = "Get-Date -Format 'HH:mm:ss'"
output = "clipboard"
```

**HTTP 요청으로 날씨 가져오기**
```toml
[[commands]]
trigger = "weather,날씨"
name = "🌤 서울 날씨"
action = "http"
url = "https://wttr.in/Seoul?format=3"
output = "clipboard"
```

**확인 후 실행 (위험한 명령)**
```toml
[[commands]]
trigger = "wipe temp"
name = "🗑 Temp 폴더 비우기"
action = "shell"
command = "Remove-Item -Path \"$env:TEMP\\*\" -Recurse -Force -ErrorAction SilentlyContinue; '완료'"
output = "clipboard"
confirm = true
```

초기 실행 시 `example.toml` 파일이 자동으로 생성되며 다양한 예시가 포함되어 있습니다.

### 기본 제공 플러그인 목록 (example.toml)

앱 최초 실행 시 자동 생성되는 `example.toml`에 포함된 명령 전체 목록입니다.

#### 🔍 동적 검색

| 트리거 | 이름 | 설명 | 사용법 |
|--------|------|------|--------|
| `google` / `구글` | 🔍 Google 검색 | Google에서 키워드 검색 | `google 리액트 훅` → 브라우저에서 Google 열기 |
| `yt` / `유튜브` | ▶ YouTube 검색 | YouTube에서 키워드 검색 | `yt lo-fi music` → YouTube 검색 결과 열기 |

## 12. 단축키 요약

### 공통

| 단축키 | 동작 |
|--------|------|
| `Ctrl+Shift+Space` | 런처 열기 / 닫기 |
| `↑` / `↓` 또는 `Ctrl+P` / `Ctrl+N` | 항목 이동 |
| `Enter` | 실행 / 클립보드에 복사 |
| `Tab` | 다음 모드로 전환 |
| `Shift+Tab` | 이전 모드로 전환 |
| `Esc` | 닫기 (미리보기 열려있으면 미리보기 먼저 닫힘) |
| `PageDown` / `PageUp` | 8개씩 이동 |

### 파일 모드 전용

| 단축키 | 동작 |
|--------|------|
| `Space` | 파일 미리보기 토글 |
| `Ctrl+C` | 선택 파일 경로 복사 |
| `Ctrl+O` | 해당 폴더 탐색기에서 열기 |
| `Ctrl+Enter` | 파일 속성 창 열기 |

### 이모지 / 즐겨찾기 모드

| 단축키 | 동작 |
|--------|------|
| `←` / `→` | 좌우 이동 |
| `↑` / `↓` | 한 행씩 위아래 이동 |
